Исследование синергетических эффектов гибридной митигации ошибок в квантовых вычислениях

Объект исследования: Методы подавления шумов (Error Mitigation) в квантовых процессорах архитектуры IBM Eagle и Falcon. Цель работы: Экспериментальное обоснование и разработка адаптивного протокола, объединяющего динамическую развязку (DD) и экстраполяцию к нулевому шуму (ZNE).
1. Теоретический базис и научная новизна

В работе реализован комплексный подход к митигации ошибок, объединяющий два различных уровня воздействия на систему:

    Физический уровень (Dynamical Decoupling): Использование последовательностей X−X в рамках стратегии планирования ALAP (As Late As Possible) для подавления фазовой декогеренции.

    Алгоритмический уровень (Zero-Noise Extrapolation): Применение линейной экстраполяции Ричардсона для аппроксимации идеального состояния системы без шумов.

Научная новизна: В отличие от статических методов, данное решение исследует эффект синергии (взаимного усиления) методов и выявляет границы их применимости в зависимости от архитектурных параметров чипа (T1​,T2​, гейтовая верность).
2. Структура программного комплекса

Программная реализация выполнена на языке Python с использованием фреймворка Qiskit 1.x.
Основные модули:

    fold_manually_stable(): Функция научно-корректного масштабирования шума через вставку унитарных пар U†U, что гарантирует сохранение логической структуры алгоритма.

    richardson_extrapolation(): Математический модуль для динамического вычисления точки нулевого шума без использования «магических» констант.

    add_dd_manual_stable(): Модуль интеграции DD-пакетов в интервалы простоя кубитов.

3. Методология эксперимента

Эксперимент проводится на трех различных типах квантовых архитектур (Fake-Backends) для верификации адаптивности:

    IBM Kyoto (27Q): Стабильная система. Тестирование точности ZNE при высоких временах когерентности.

    IBM Sherbrooke (127Q): Крупномасштабная система со средним уровнем шума. Поиск баланса между DD и ZNE.

    IBM Brisbane (127Q): Система с критическим уровнем шума. Исследование «порога выживаемости» сигнала.

4. Результаты и выводы (Summary)

В ходе серии запусков (10,000 shots на итерацию) были получены следующие ключевые результаты:

    Подтверждена гипотеза о «Гейтовом оверхеде»: На стабильных чипах (Kyoto) избыточное применение DD может снижать точность, что делает ZNE приоритетным методом.

    Выявлена синергия: На средних архитектурах гибридный метод эффективно компенсирует аппаратные ошибки самих DD-импульсов с помощью ZNE-фильтрации.

    Обоснована необходимость адаптивности: Разница в поведении бэкендов доказывает, что универсального метода не существует, что является фундаментом для разработки адаптивного алгоритма выбора стратегии (Глава 4).
